# Pruebas Técnicas para Java con Respuestas y Ejemplos

## Junior

### Pregunta 1
**Escribe un programa que imprima los números del 1 al 10.**

**Respuesta:**
```java
public class Main {
    public static void main(String[] args) {
        for (int i = 1; i <= 10; i++) {
            System.out.println(i);
        }
    }
}
